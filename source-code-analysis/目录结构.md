jsplumb 的源码都在 src 目录下，其目录结构如下。

````
src
├── jsPlumb           # 核心
├── defaults          # the default Connectors, Endpoint and Overlay definitions.
├── anchors.js        # 锚点相关操作 
├── endpoint
├── overlay-component           # overlay 操作
├── connection         
├── connectors-flowchart             
├── dom.jsPlumb          # drag/drop 相关操作
├── render-svg           # 渲染 svg
├── dom-adapter          # 原生 Dom 操作
├── base-library-adapter.js            # 库适配器基类？？ 
├── group           # 分组 ？？
├── util           # 工具
├── browser-util    # 浏览器专属工具函数   
````
